Class: TextWalker

text-walker. TextWalker

new TextWalker(root, filter)

The TextWalker class provides a seekable view over a Node tree.

Parameters:
Name Type Description
root Node

The root Node of the tree to consider
filter Object

An Object containing a key, "acceptNode", that has a function value that accepts a Node and returns true or false to indicate whether the Node should be included when traversing the tree and counting text

Seek the TextWalker to a new text offset.

Parameters:
Name Type Description
whence Symbol

one of SEEK_SET, SEEK_CUR or SEEK_END. The meaning is the same as for POSIX lseek(2) except that it is impossible to seek past the end of the text
